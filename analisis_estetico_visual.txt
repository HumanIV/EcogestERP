accesibilidad. Los errores de código traducidos a problemas visuales son:

- Problemas de Layout en móviles

GUÍA DE DISEÑO RECOMENDADA - SISTEMA "ECOGEST":
================================================================================

COLORES PRINCIPALES:
- Primary Eco: #10B981 (verde esmeralda)
- Secondary: #059669 (verde oscuro)
- Accent: #F59E0B (ámbar/warning)
- Danger: #EF4444 (rojo)
- Background: #F9FAFB (gris claro)
- Surface: #FFFFFF (blanco)

TIPOGRAFÍA:
- Headers: "Montserrat", sans-serif
- Body: "Inter", system-ui, sans-serif
- Data/Numbers: "JetBrains Mono", monospace

ESPACING (sistema 8px):
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px

COMPONENTES:
- Border Radius: 8px (sm), 12px (md), 16px (lg)
- Shadows: 0 1px 3px (sm), 0 4px 6px (md), 0 10px 20px (lg)
- Buttons: min-height 44px (accesibilidad touch)
