2. JUSTIFICACIÓN DE SEGURIDAD Y ARQUITECTURA
================================================================================

PROBLEMA:
- Dar acceso a ciudadanos al Admin = VULNERABILIDAD CRÍTICA
- El Admin contiene: RRHH, Financiera, Auditoría, cuadrillas, etc.
- Usuarios externos NO pueden tocar NADA de eso

SOLUCIÓN:
- "Subir" formularios públicos (denuncias, permisos, licencias) a Ecogest
- Ecogest Web = React moderno (no HTML estático)
- App Móvil = Solo denuncias (lo más usado, rápido)
- Admin = SOLO empleados (oficina completa que se auto-gestiona)

FLUJO DE DATOS:
App Móvil (denuncias) → Backend API ← Ecogest Web (denuncias+permisos+licencias)
                              ↓
                     Admin CoreUI (gestiona todo + se auto-gestiona)
